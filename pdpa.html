ขั้นตอนที่ 1 / 3 : ตรวจสอบเงื่อนไขข้อตกลงก่อนการสั่งซื้อโปรแกรม
เงื่อนไข/ข้อตกลงสำหรับเข้าใช้งานโปรแกรม
ADTSpreadsheet
ในเงื่อนไข/ข้อตกลงต่อไปนี้ ให้ผู้เข้าใช้งานโปรแกรมทำความเข้าใจอย่างละเอียดและผู้เข้าใช้งานโปรแกรม ADTSpreadsheet ต้องปฏิบัติและยินยอมตามข้อตกลงในเอกสารฉบับนี้ทุกประการ ดังนี้: ข้อที่ 1 : โปรแกรม ADTSpreadsheet เป็นซอฟต์แวร์ที่พัฒนาบน Microsoft Excel โดยใช้ความรู้ด้านวิศวกรรมและ Visual Basic for Applications เพื่อช่วยจัดทำเอกสารรายการคำนวณทางวิศวกรรม โดยอยู่ภายใต้ดุลยพินิจของผู้ใช้ ข้อที่ 2 : ผู้ใช้งานจะต้องกรอกข้อมูลจริงในการลงทะเบียน หากตรวจพบว่าข้อมูลเป็นเท็จ ผู้พัฒนาขอสงวนสิทธิ์ในการบล็อกการใช้งานโดยไม่ต้องแจ้งล่วงหน้า ข้อที่ 3 : ผู้ใช้งานต้องรับผิดชอบต่อเอกสารที่สร้างจากโปรแกรม หากเกิดปัญหาใด ๆ ที่ไม่เกี่ยวข้องกับโปรแกรม ถือเป็นความรับผิดชอบของผู้ใช้งานแต่เพียงผู้เดียว ข้อที่ 4 : ห้ามดัดแปลง แก้ไข คัดลอก หรือเปลี่ยนแปลงโครงสร้างโปรแกรมไม่ว่ากรณีใด ๆ ข้อที่ 5 : หากพบการโจมตีหรือพฤติกรรมทุจริต ผู้พัฒนาขอสงวนสิทธิ์ในการยกเลิกสิทธิ์การใช้งานทันที ข้อที่ 6 : การทดลองใช้งาน 7 วันนับจากวันลงทะเบียน โดย 1 User ID ใช้งานได้เพียง 1 เครื่อง หากพบการใช้งานซ้ำจะถูกระงับทันที ข้อที่ 7 : ผู้ใช้งานยินยอมให้จัดเก็บข้อมูลส่วนบุคคลเพื่อระบุตัวตน ได้แก่ ชื่อ-นามสกุล, เบอร์โทร, เลขบัตรประชาชน, LINE ID, IP, Machine ID เพื่อใช้ควบคุมสิทธิ์การใช้งานเท่านั้น ข้อที่ 8 : การบริการหลังการขายครอบคลุมเฉพาะปัญหาที่เกี่ยวกับตัวโปรแกรมเท่านั้น ปัญหาจากสภาพแวดล้อมของผู้ใช้อาจมีค่าบริการเพิ่มเติม
