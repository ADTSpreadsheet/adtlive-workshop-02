มาตรฐานการออกแบบ : มาตรฐานว.ส.ท. (ISBN 974-858-145-4 E.I.T. Standard 1007-34 )
ข้อกำหนดการออกแบบ : กฎกระทรวง กำหนดการออกแบบโครงสร้างอาคารและลักษณะและคุณสมบัติของวัสดุในงานโครงสร้างอาคาร
วิธีการออกแบบ : หน่วยแรงใช้งาน (Working Stress Design / WSD )

1. งานฐานรากอาคาร
- โปรแกรมออกแบบฐานราก ค.ส.ล. แบบฐานแผ่ และฐานเสารวม
- โปรแกรมออกแบบฐานราก ค.ส.ล. แบบเสาเข็ม (Footing F2,F3,F4,F5)

2. งานออกแบบเสาอาคาร
- เสากลม เสาเหลี่ยม สั้น/ยาว

3. งานออกแบบพื้นอาคาร
- พื้น Two Way และ One Way
- พื้นแผ่นสำเร็จรูป (Plank Slab)

4. งานออกแบบคาน
- วิเคราะห์ FBD, SFD, BMD + Three Moment Equation
- หน้าตัดคาน ปรับ Re-bar ได้

5. งานออกแบบบันได
- Type A (ชานพักวางบนคาน), Type B (ชานพักยื่น)

6. งานออกแบบหลังคา
- ทรงจั่ว / โมเดิร์น / ปั้นหยา

7. รายการอื่น ๆ
- Block รายการคำนวณ
- Block ข้อกำหนดน้ำหนักบรรทุก
- คู่มือใช้งาน
- ระบบ ADTLine-Bot
- ระบบ Live สด ตามตาราง
